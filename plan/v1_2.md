# v1.2 高级策略与生态

## 🎯 目标
支持大规模训练与生物领域深度功能。

## 📅 新增功能

### 1. 高级训练策略
- [ ] 自动混合精度（AMP）：支持 fp16 训练。
- [ ] 单机多卡（DataParallel）：使用多块 GPU。
- [ ] 多机分布式（DDP）：探索跨 Worker 分布式训练。

### 2. 生物领域特性
- [ ] 结构可视化：集成 Mol* / NGL Viewer 查看 PDB。
- [ ] 预训练模型：内置 ESM-2 等，支持微调。

### 3. 系统运维
- [ ] 节点自动扩缩容：对接云厂商 API，按队列自动开关机。
- [ ] 告警通知：训练结束或失败时通知（邮件/Slack/钉钉）。
- [ ] 模型版本对比：页面对比不同版本指标（Diff View）。
