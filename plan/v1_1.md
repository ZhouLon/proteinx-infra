# v1.1 体验优化与可视化

## 🎯 目标
在 v1.0 跑通的基础上，提升用户体验和可观测性。

## 📅 新增功能

### 1. 实验可视化
- [ ] 实时曲线：接入 WebSocket，实时绘制 Loss / Accuracy。
- [ ] TensorBoard：在 Master 集成，查看训练细节。
- [ ] 资源监控：显示 Worker 的 GPU/CPU/内存使用。

### 2. 高级数据管理
- [ ] 预处理流：在子模型配置中增加预处理环节（截断、填充、编码）。
- [ ] 数据校验：上传时自动格式检查（FASTA/PDB）。

### 3. 任务管理增强
- [ ] 任务取消：Web 端中断正在运行的训练。
- [ ] 断点续训：从上一次 Checkpoint 继续训练。
- [ ] 多任务排队：允许提交多个任务，Worker 依次执行。

### 4. 前端体验
- [ ] 表单化配置：用表单替代 JSON 编辑。
- [ ] 移动端适配：优化手机浏览器体验。

### 5. 项目管理与首页优化
- [ ] Dashboard 首页性能优化（大数据量场景不卡顿）。
- [ ] 项目首页支持新建项目与概要统计（模型数量）。
- [ ] 工作目录下建立 `projects/` 结构，持久化项目元信息。
- [ ] 项目详情页（特别页）：删除（需密码确认）、改名与介绍编辑。
